Ruby Hash vs. Javascript Objects

Hashes and objects are very similar data structures. They even look alike.
jim = { name: "Jim", age: 25, shoe_size: 9 }
var Jim = { name: "Jim", age: 25, shoe_size: 9 };
But there is one distinct difference between a ruby hash and a javascript object. Javascript objects can take a function as a value. Since methods in Ruby aren't objects, you can't assign a method as a hash value. This is a handy bit of functionality possessed by JavaScript objects, with no direct equivalent for Ruby hashes. If you wanted to achieve something similar in Ruby you would need to use classes.
